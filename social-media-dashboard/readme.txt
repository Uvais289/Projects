## Description

This project showcases a landing page built with HTML, CSS, and JavaScript.  The landing page includes a header, a hero section with a dynamic background, features section, and a footer. The hero section allows users to input their preferences and dynamically changes the background color based on their input.

## Functionalities

- **Dynamic Background:** Users can input their preferences in a form, and the background color of the hero section dynamically changes based on their input.
- **Feature Sections:** The landing page includes sections highlighting different features.
- **Change Background Color Button:** Users can click a button to change the background color of the hero section randomly.
